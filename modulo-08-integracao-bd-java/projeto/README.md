# 🎯 Módulo 08 - Integração Java com Banco de Dados

## ✅ Desafios deste módulo
- ✅ Desafio Projeto

## 📖 Descrição
Projeto que conecta aplicação Java com banco de dados relacional usando JPA ou JDBC.

## ▶️ Como executar
```
mvn clean install
mvn spring-boot:run
```

## 💡 Observações
- Inclui configuração de banco (H2 ou outro).
